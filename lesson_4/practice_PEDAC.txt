# PROBLEM:

# Given a string, write a method `palindrome_substrings` which returns
# all the substrings from a given string which are palindromes. Consider
# palindrome words case sensitive.

# Test cases:

# palindrome_substrings("supercalifragilisticexpialidocious") == ["ili"]
# palindrome_substrings("abcddcbA") == ["bcddcb", "cddc", "dd"]
# palindrome_substrings("palindrome") == []
# palindrome_substrings("") == []

Input: a string
Output: an array of string

#RULES:
# Explicit requirement:
  - Return all the substrings that a palidromes (Case sensitive - "Tit" is not a palidromes, "tit" is")
  - All of the palidromes are pushed into an array
  - Return empty array if there is no palidromes
  - Return empty array if there is no input
  
# Implicit requirement:
  - Does the method mutate the input or returns the copy of it
  - Does the method check if the input is valid (In this case, input has to be a string")
